# ztrax Community Themes

Optional themes for ztrax. These are not built-in — copy any you like into
your local themes directory and select them in your config.

## Install

```sh
# macOS / Linux
cp <theme>.toml ~/.config/ztrax/themes/
```

Then set it in `~/.config/ztrax/config.toml`:

```toml
theme = "<theme-name>"
```

Or press `t` in ztrax to cycle through all discovered themes.

See [../README.md](../../README.md) for the full theme-file reference.
